# git_cmd

### Initialize git

```
git init
```

### Push to remote repo (github)

1. ` git add.`
2. `git commit -m "Message"`
3. `git push`

## Status

`git status`

## BRANCH

Ceate new branch

```
git branch branch_name
```

```
git checkout branch_name
```

```
git branch -d branch_name
```

List branches

```
git branch
```

Remote branch

```
git push origin branch_name
```

merge branch
`git merge branch_name`
