# git_commit

Initialize git
```
git init
```
push to remote repo (github)

1. ``` git add.```
2. ``` git commit -m "Message" ```
3. ``` git push ```

## Status

``` git status ```

## BRANCH

create new branch
``` git branch branch_name ```
switch brach
``` git checkout branch_name ``
delete branch
``` git branch -d branch_name ```
list branches
``` git branch ```
remote branch
``` git push origin branch_name ```
merge branch
``` git merge branch_name ```
